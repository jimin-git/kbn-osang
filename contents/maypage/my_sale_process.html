닫기

연구책임자
(실무담당자) 분양지원센터
상담자 단위은행
담당자

  • 1. 작성중

    분양신청서 작성 중 임시저장한 상태 (분양신청서 수정가능)

  • 2. 신청완료

    분양신청서를 KBN 담당자에게 제출한 상태 (자원수정을 제외하고 서류수정가능)

    *KBN 담당자가 단위은행에 자원유무 확인 > 실무담당자에게 단위은행 안내 > 서류검토 > 단위은행 지정

  • 3. 접수대기

    상담자가 작성자에게 책임연구자 서명요청 메일 전송(서류수정 불가능)

  • 4. 접수완료

    분양신청서에 연구책임자 서명이 완료된 상태

  • 5. 심의대기

    단위은행에서 심의 일자가 지정되어 분양위원회에서 심의 중인 상태

  • 6. 심의완료

    단위은행에서 분양심의가 완료 (심의의견을 볼 수 있음)
    • 승인: 자원 분양을 받을 수 있는 상태
    • 반려: 보완후재신청 버튼을 눌러 신청서 수정 후 다시 진행
    • 추가분양신청 : 자원을 추가할 경우 추가분양신청 버튼을 눌러 신청서 수정 후 다시 진행
    • 이용계획변경신청: 연구기간, 이용기간, 연구자 변경을 할 경우 이용계획변경 버튼을 눌러 신청 수정 후 다시 진행

  • 7. 분양완료

    인체자원 분양이 완료된 상태 (이용기간 종료 후에는 인체자원 폐기)
    * 분할분양을 하는 경우 심의완료로 표기됩니다.

  • 8. 활용종료

    분양자원 활용기간이 종료되면 단위은행에서 활용종료일을 지정
